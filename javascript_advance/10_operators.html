JavaScript Operators

JavaScript operators are used to perform particular actions.

They can be used to assign values, compare values, combine values, perform arithmetic operations and more.

JavaScript opertors are symbols (or set of symbols) or keywords.

There are different types of operators.

1. Assignment Operators

2. Arithmetic Operators

3. String Operators

4. Comparision Operators

5. Logical Operators

6. Bitwise Operators

Assignment Operators

The assignment operator is used to assign a value to a variable.

It uses the equal = symbol.
