JavaScript Data Types

In JavaScript or in any programming language, the concept of data types is important.

What are Data Types?

Data Types are the different kinds of data that can be strored or used in a JavaScript program.

Primitive Data

Primitive data are data that are not object and has no method.

There are 5 basic primitive data types.

1. String

2. Number

3. Boolean

4. Undefined

5. Null

String Data Type

The string data type is a sequence of characters used to represent text.

Strings can be written with either single or double quotes.

You can use which one you like.

Number Data Type

The number data type is an integer or a floating point number (number with decimal).

Boolean Data Types

The boolean data types is a logical data type that can only have true or false as values.

Booleans are mainly used for conditional testing.

The Undefined Data Type

If a variable has no assigned value, the value is undefined.

In the example, the title variables is declared but is not assigned to any value.

Therefore, the value is undefined.

The Null Data Type

The null data type is a special data type denoting a null value - meaning it's not an empty string " " or 0.

It simply means nothing
