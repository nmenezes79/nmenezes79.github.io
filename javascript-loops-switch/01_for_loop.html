JavaScript For Loops

Explanation:

  • Initialization (let i = 1): The loop starts with i being initialized to 1.
  • Condition (i <= 5): The loop will continue running as long as i is less than or equal to 5.
  • Increment (i++): After each loop iteration, the value of i is increased by 1.
  • Loop body (console.log(i)): In each iteration, it prints the current value of i to the console.
