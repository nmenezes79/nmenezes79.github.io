35_promise_chaining

In this example, asyncFunction is a function that returns a promise which resolves after 1 second with the value multiplied by 2.

We chain multiple calls to asyncFunction, each time passing the result of the previous call to the next one.

Finally, we handle any errors that might occur during the execution of the promises using the catch method.

When you run this code, you'll see each result printed in sequence after each promise resolves.
