36_async

We define a function fetchData that returns a Promise. Inside the Promise constructor, we simulate an asynchronous operation with setTimeout, and once the operation is done, we resolve the Promise with some data.

We define an async function getData that uses the async keyword, allowing us to use await inside it. Inside this function, we call fetchData using await, which waits for the Promise returned by fetchData to resolve. Once resolved, we log the fetched data.

Finally, we call the getData function to initiate the fetching process.
